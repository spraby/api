# Исправление ошибки 404 при логауте пользователя

Дата создания: 2025-12-28

## Описание

## Найденная проблема
В nav-user.tsx используется путь `/logout`, но маршрут находится в группе с префиксом `sb/admin`, поэтому правильный путь: `/sb/admin/logout`

- [x] Раздел 3: Реализация исправления
  - [x] Исправить маршрут logout (маршрут был правильный в routes/web.php)
  - [x] Обновить компонент nav-user.tsx (изменен путь с `/logout` на `/sb/admin/logout`)
  - [x] Обработка ошибок (не требуется - стандартная обработка Inertia.js)

- [ ] Раздел 4: Тестирование
  - [ ] Проверить logout в браузере
  - [ ] Проверить редирект после logout
  - [ ] Проверить очистку сессии

## Решение
**Изменение**: `resources/js/admin/components/nav-user.tsx:40`
```typescript
// Было:
router.post('/logout')

// Стало:
router.post('/sb/admin/logout')
```
