# Исправление переключения тем (светлая/темная)

Дата создания: 2025-12-20
Статус: **ВЫПОЛНЕНО**

## Описание
Темы (светлая и темная) не везде корректно переключаются. Проблемы обнаружены в нескольких компонентах.

## Внесённые изменения

### 1. theme.css - добавлены dark mode стили для:
- `.p-popover` - фон, границы, тени
- `.p-menu` - пункты меню, разделители, hover эффекты
- `.p-sidebar` - drawer/sidebar компонент
- `.p-button.p-button-text` - текстовые кнопки
- `.p-button.p-button-secondary.p-button-text` - secondary текстовые кнопки
- `.p-avatar` - аватар компонент

### 2. AdminSidebar.vue - улучшены стили popup подменю:
- Добавлены явные dark mode стили для ссылок
- Исправлены hover эффекты для тёмной темы
- Добавлены стили для активных элементов
- Исправлены стили градиентного заголовка

### 3. ThemeToggle.vue - проверен
- Tailwind классы с `dark:` вариантами работают корректно
- Фон Popover теперь правильно стилизован через theme.css
